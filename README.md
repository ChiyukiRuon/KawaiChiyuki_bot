## 机器人功能
### Pixiv
- `/[day|week|month]_rank` 获取`Pixiv[日|周|月]榜`的图片
- `/[day|week]_rank_r18` 获取`Pixiv R18[日|周]榜`的图片
- `/rank_g18g` 获取`Pixiv R18G排行榜`的图片
> 在上述代码后加上页数可以获取更多图片,一页5张，共6页。不加默认获取排行榜前五张  
> 例：`/day_rank 2`  
> 获取Pixiv日榜第6-10张图片  
> 若页数为负数则是倒序发送
- `/sese` 随机获取Pixiv的图片
### 一言
- `/yan` 一言二次元语录
