## 机器人功能
### Pixiv
- /[day|week|month]_rank 获取Pixiv[日|周|月]榜的图片
- /[day|week]_rank_r18 获取Pixiv R18[日|周]榜的图片
- /rank_g18g 获取Pixiv R18G排行榜的图片
